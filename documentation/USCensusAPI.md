# US Census API Documentation

This document provides an overview of how to use the US Census Bureau's API to access various datasets, including the American Community Survey (ACS). The API allows users to query demographic, social, economic, and housing data for different geographic levels.

## API Manual

For detailed information on the API endpoints, parameters, and response formats, please refer to the official US Census API User Guide: [Census API User Guide](https://www2.census.gov/data/api-documentation/api-user-guide.pdf)
